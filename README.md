# Education Projects

A personal hands-on repository where I build mini projects while learning modern technologies across data engineering, analytics, cloud, AI, and application development.

## About This Repository

This repo is a collection of small, practical projects created to learn by doing.  
Instead of only reading concepts, I use this space to experiment, build, break, fix, and understand how different tools work in real scenarios.

The goal is to develop working knowledge of technologies through implementation-focused learning.

## What You’ll Find Here

This repository may include mini projects related to:

- Python
- SQL
- Data Engineering
- Snowflake
- dbt
- Airflow
- AWS
- Streamlit
- APIs
- AI / LLM / RAG systems
- Automation scripts
- Analytics and dashboarding concepts

Each project is intended to focus on learning a specific concept, tool, workflow, or integration.

## Purpose

The purpose of this repository is to:

- practice new technologies in a hands-on way
- build small working implementations
- strengthen technical fundamentals
- document progress over time
- create a reusable reference for future projects
- gradually move from concepts to real-world applications
